  1. Funcionalidades Implementadas                                                                                                                                                                        
                                                                                                                                                                                                          
  Core Features                                                                                                                                                                                           
                                                                                                                                                                                                          
  1.1 Gestión de Proyectos (ProjectManager)                                                                                                                                                               
  - Crear, activar y listar proyectos                                                                                                                                                                     
  - Estructura de carpetas en ~/Documents/KokoroLibrary                                                                                                                                                   
  - Sistema de numeración secuencial automática (001.txt, 001.wav, 001.png)                                                                                                                               
  - Búsqueda y navegación por proyectos                                                                                                                                                                   
  - Límite de 30 archivos recientes por proyecto                                                                                                                                                          
                                                                                                                                                                                                          
  1.2 Captura y OCR                                                                                                                                                                                       
  - Captura de pantalla (Cmd+Alt+A): Screenshot interactivo + OCR                                                                                                                                         
  - Batch de imágenes: Procesar múltiples imágenes con OCR                                                                                                                                                
  - PDF a Audio: Extrae imágenes del PDF (con poppler) → OCR → TTS                                                                                                                                        
  - EPUB/MOBI a Audio: Renderiza páginas del ebook con PyMuPDF → OCR → TTS

  - Sistema de voces personalizado (mezcla de voces)                                                                                                                                                      
  - Ajuste dinámico de velocidad según longitud del texto                                                                                                                                                 
  - Filtro de audio tipo "radio" para mejorar calidad

    ↓                                                                                                                                                                                                     
  Kokoro TTS (phonemes → audio)                                                                                                                                                                           
    ↓                                                                                                                                                                                                     
  Apply radio filter                                                                                                                                                                                      
    ↓                                                                                                                                                                                                     
  Save: ID.txt + ID.wav + ID.png                                                                                                                                                                          
    ↓                                                                                                                                                                                                     
  Play audio + Update menu

    ↓

  TTS + Save + Play

    ↓

    ↓ OCR                                                                                                                                                                                                 
    ↓ TTS

    ↓ Create Fusion folder

  Desafíos Técnicos                                                                                                                                                                                       
                                                                                                                                                                                                          
  1. Kokoro TTS en Swift: No existe port nativo. Opciones:                                                                                                                                                
    - Usar AVSpeechSynthesizer (menor calidad)                                                                                                                                                            
    - Integrar Kokoro ONNX con CoreML (complejo)                                                                                                                                                          
    - Llamar al ejecutable Python desde Swift (híbrido)                                                                                                                                                   
    - Usar APIs cloud (OpenAI, ElevenLabs) según CLAUDE.md                                                                                                                                                
  2. EPUB Processing: PDFKit no lee EPUB nativamente. Alternativas:                                                                                                                                       
    - Usar WebKit para renderizar HTML del EPUB                                                                                                                                                           
    - Extraer HTML y aplicar OCR                                                                                                                                                                          
    - Buscar library de terceros (EPUBKit)                                                                                                                                                                
  3. Voice Blending: La mezcla matemática de voces (santa*0.7 + alex*0.3) requiere acceso a embeddings. AVSpeechSynthesizer no lo permite. Solo viable si se porta Kokoro.                                
  4. Global Hotkeys: CGEventTap requiere permisos de Accessibility en macOS (sandbox challenge para App Store).                                                                                           
                                                                                                                                                                                                          
  Recomendaciones                                                                                                                                                                                         
                                                                                                                                                                                                          
  1. Comenzar con MVP usando AVSpeechSynthesizer para validar arquitectura                                                                                                                                
  2. Migrar a Kokoro o API cloud en Fase 2 si se necesita calidad superior                                                                                                                                
  3. Usar Vision framework para OCR (mejor que soluciones de terceros)
